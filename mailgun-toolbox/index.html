Mailgun Toolbox

Mailgun email monitoring dashboard for Rock RMS

The Mailgun Toolbox plugin provides a comprehensive email health dashboard inside Rock RMS. Monitor delivery rates, manage bounces and complaints, drill into failure details, and analyze per-domain statistics — all without leaving Rock.

Requires Rock RMS v16.9+ (Obsidian) and a Mailgun account with API access.

Dashboard tabs

  • Overview: Delivery, open, click, and bounce rate cards with totals. Line charts for daily counts, failure rate, and engagement rates. Clickable complaint count.
  • Details: Permanent vs. temporary failure breakdown, top error codes by frequency, and the 25 most recent failed delivery events.
  • Bounces: Paginated suppression list with search-by-email and per-row delete to remove addresses from Mailgun.
  • Historicals: Time-series data at hourly, daily, or monthly resolution. Click any failed count to drill into specific failure events.
  • Domains: Per-recipient-domain breakdown of sent, delivered, hard-failed, and soft-failed counts with success rate percentages.

A global date range filter (Current Month, Previous Month, Last 7/30/90 Days, Custom) applies across all tabs. The default range is configurable per block instance.

Quick start (development)

# C# Backend
dotnet build com.dtschurch.ManagedMail/com.dtschurch.ManagedMail.csproj -c Release

# Obsidian Frontend
cd com.dtschurch.ManagedMail.Obsidian
npm install
npm run build

# Package as .plugin file
dotnet tool install --global sparkdevnetwork.rock.devtool
rock-dev-tool plugin pack

Block actions

Action Purpose
GetStats Aggregate statistics for the selected date range.
GetTimeSeriesStats Time-series data at hour/day/month resolution.
GetBounces / GetBouncesPage Paginated bounce suppression list.
DeleteBounce Remove an address from the Mailgun bounce suppression list.
SearchBounce Look up a specific bounce address.
GetEventsPage Paginated event log (failures, complaints, deliveries).
GetComplaints / DeleteComplaint View and manage the complaint suppression list.
GetDomainBreakdown Per-recipient-domain delivery statistics.

Mailgun API endpoints used

Method Endpoint Purpose
GET /v3/{domain}/stats/total Aggregate and time-series statistics
GET /v3/{domain}/bounces Bounce suppression list
DELETE /v3/{domain}/bounces/{address} Remove a bounce
GET /v3/{domain}/events Event log (failures, complaints)
GET /v3/{domain}/complaints Complaint suppression list
DELETE /v3/{domain}/complaints/{address} Remove a complaint
