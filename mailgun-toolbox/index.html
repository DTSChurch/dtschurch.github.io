Mailgun Toolbox

Email monitoring dashboard for Rock RMS

The Mailgun Toolbox plugin provides an email health dashboard inside Rock RMS. Monitor delivery rates, manage bounces and complaints, drill into failure details, and analyze per-domain statistics — all without leaving Rock.

Requires Rock RMS v16.9+ (Obsidian) and a Mailgun account with API access.

Setup Guide Dashboard Guide

What you get

  • Overview: See delivery, open, click, and bounce rates at a glance with daily trend charts.
  • Details: Investigate delivery failures — see error breakdowns and the most recent failed messages.
  • Bounces: View and manage the list of bounced email addresses. Remove addresses to allow re-delivery.
  • Historicals: Explore email volume over time at hourly, daily, or monthly resolution.
  • Domains: See how email performs per recipient domain (gmail.com, outlook.com, etc.).

A date range filter applies across all tabs. You can choose from preset ranges or enter custom dates.

Getting started

  1. Install the plugin — upload the plugin file and restart Rock.
  2. Configure your Mailgun credentials — set your API key and sending domain.
  3. Explore the dashboard — learn what each tab shows and what to look for.
