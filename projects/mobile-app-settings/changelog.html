Version history

Release notes and changes for the Mobile App Settings plugin.

v1.0.0

Initial release

  • Admin dashboard with hierarchical category tree and settings grid
  • Full CRUD for categories with recursive sub-category support
  • Settings management using Rock’s native Attribute system
  • Support for all Rock field types (Text, Boolean, Date, Integer, etc.)
  • Per-site isolation — each mobile app site has its own settings tree
  • Custom AppSetting Lava filter for accessing settings in templates
  • Built-in Lava example generator with click-to-copy snippets
  • Inline value editing with Rock’s AttributeValuesContainer
  • Two database migrations for block type registration and admin page creation
