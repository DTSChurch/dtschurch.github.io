Jobs & Automation

Rock Jobs that run on a schedule to handle appointment reminders, follow-up communications, and workflow triggers.

Send Appointment Reminder Communication

Sends reminder emails and/or SMS messages to people with upcoming appointments.

Configuration

AttributeDescription
Send Reminders Comma-delimited list of days before an appointment to send a reminder. Example: 2,4 sends reminders 2 and 4 days before.

Behavior

  • Queries appointments within the configured day range from today
  • Excludes cancelled appointments — only Pending and Attended statuses receive reminders
  • Uses the Reminder System Communication configured on each appointment type
  • Supports both email and SMS channels
  • Skips appointment types without a reminder communication configured

Merge Fields (in System Communication)

FieldDescription
{{ Appointment }}The appointment entity with all navigation properties.
{{ Person }}The person the appointment is booked for.

Send Appointment Completed Notification

Sends follow-up communications for appointments that have passed, based on attendance status.

Configuration

AttributeDescription
System Email The system communication template to use for notifications.
Days Prior (Pending) Number of days after an appointment to notify contacts about Pending (no-show) appointments.
Days Prior (Attended) Number of days after an appointment to notify contacts about Attended appointments.

Behavior

  • Groups appointments by appointment type and contact
  • Sends a single summary email per contact with all relevant appointments
  • Only processes appointments with Pending or Attended status within the configured day ranges
