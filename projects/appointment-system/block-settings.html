Block Settings

All configurable attributes for the Book Appointment block, listed in the order they appear in Rock's block settings panel.

Setting Type Default Description
Appointment Type Appointment Type Picker None If set, locks the block to a single appointment type and skips step 1. Uses the custom AppointmentTypeFieldType tree picker.
Phone Number Required Boolean False When enabled, the phone number field on step 3 becomes required.
Connection Status Defined Value Prospect The connection status assigned to new person records created by this block.
Record Status Defined Value Active The record status assigned to new person records created by this block.
Appointment Type Card Template Lava (Code Editor) See Lava Templates Lava template for each appointment type card. Controls what renders inside the card container.
Slot Card Template Lava (Code Editor) See Lava Templates Lava template for each time slot pill inside the date accordion.
Confirmation Template Lava (Code Editor) See Lava Templates Lava template for the success screen after booking.
Header Template Lava (Code Editor) Empty Optional Lava template rendered above the booking flow for branding or instructions.
Custom CSS CSS (Code Editor) See CSS Reference CSS injected via a <style> tag. Styles the Lava-rendered card content. Override to match your theme.
Show Campus Filter Boolean False When enabled, displays a campus dropdown filter on the appointment type selection screen.
Show Location Filter Boolean False When enabled, displays a location dropdown filter. Options cascade from the selected campus.
Show Panel Boolean True When enabled, wraps each step in a Rock Panel with a header. Disable for a cleaner embedded look.
