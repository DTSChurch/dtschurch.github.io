Customization

Lava Templates

The Book Appointment block uses Lava templates for card rendering, confirmation screens, headers, and CSS. Each template is resolved server-side with entity-rich merge fields and injected into the Vue component.

Appointment Type Card Template

Renders the content of each appointment type card on the selection screen (Step 1). The block manages the grid layout, click handling, and disabled state — this template only controls what appears inside each card.

Merge Fields

FieldTypeDescription
{{ AppointmentType.Title }}StringThe appointment type name.
{{ AppointmentType.Campus }}Campus EntityFull Rock Campus entity. Access .Name, .ShortCode, etc.
{{ AppointmentType.Location }}Location EntityFull Rock Location entity. Access .Name, .Street1, etc.
{{ AppointmentType.AvailableSlotsCount }}IntegerTotal number of bookable slots across all schedules.
{{ AppointmentType.IdKey }}StringThe entity's obfuscated IdKey.
{{ AppointmentType.Guid }}GuidThe entity's Guid.
{{ CurrentPerson }}Person EntityThe currently logged-in person (null if anonymous).

Slot Card Template

Renders the content of each time slot pill inside the date accordion (Step 2). The block manages the grid layout and click handling.

Merge Fields

FieldTypeDescription
{{ Slot.Time }}StringFormatted time (e.g., "9:00 AM" or "9:00 AM - 9:30 AM" for subdivided slots).
{{ Slot.SlotsAvailable }}String"Unlimited" or a number string (e.g., "3").
{{ Slot.SlotsAvailableCount }}IntegerNumeric remaining count (for conditional styling).
{{ Slot.Date }}StringDate in yyyy-MM-dd format.
{{ Slot.Schedule }}Schedule EntityFull AppointmentSchedule entity.
{{ CurrentPerson }}Person EntityThe currently logged-in person.

Default Template

[Placeholder: Screenshot of default slot pill] 

<div class="slot-pill-time">{{ Slot.Time }}</div>
<div class="slot-pill-availability{% if Slot.SlotsAvailable == 'Unlimited' %} open{% elseif Slot.SlotsAvailableCount <= 2 %} scarce{% elseif Slot.SlotsAvailableCount <= 5 %} limited{% endif %}">
    {% if Slot.SlotsAvailable == 'Unlimited' %}Open{% else %}{{ Slot.SlotsAvailable }} left{% endif %}
</div>

Confirmation Template

Renders the success screen after a booking is confirmed (Step 4).

Merge Fields

FieldTypeDescription
{{ Appointment }}Appointment EntityThe booked appointment (when available).
{{ AppointmentType }}AppointmentType EntityThe appointment type (when predetermined).
{{ Person }}Person EntityThe person who booked.
{{ CurrentPerson }}Person EntityThe logged-in user.

Header Template

Renders HTML above the entire booking flow. Useful for branding, instructions, or context-specific messaging.

Merge Fields

FieldTypeDescription
{{ AppointmentType }}AppointmentType EntityThe predetermined type (null if not set).

Custom CSS

The Custom CSS block attribute injects a <style> tag into the page. The default value provides all card and slot styling. Override any class to match your theme. Key CSS classes:

ClassElement
.appointment-type-cardOuter card container (flex, border, hover)
.appointment-type-card-accentLeft color accent bar
.appointment-type-card-bodyCard content area
.appointment-type-titleCard title
.appointment-type-campus-badgeCampus pill badge
.appointment-type-locationLocation line with icon
.appointment-type-slots-availableGreen slot count
.appointment-type-slots-noneMuted "No slots" text
.appointment-type-cta"Schedule" call-to-action
.appointment-type-card--disabledDisabled card state (no slots)
.slot-pillTime slot card container
.slot-pill-timeSlot time text
.slot-pill-availabilitySlot availability text
.slot-pill-availability.scarceRed (2 or fewer left)
.slot-pill-availability.limitedAmber (5 or fewer left)
.slot-pill-availability.openGreen (unlimited)
