Admin Blocks

The internal-facing Obsidian blocks for managing appointment types, schedules, appointments, contacts, and calendar views.

Appointment Type Detail

[Placeholder: Screenshot of appointment type detail edit mode]

Full configuration for an appointment type:

  • General: Name, appointment term (customizable label), category, active status
  • Location & Scheduling: Campus picker, location picker, schedule card picker (with rich preview showing recurrence, next occurrence, slot capacity, and HTML descriptions), weeks ahead, booking threshold
  • Communications: Confirmation and reminder system communication pickers
  • Automation: Booked and rescheduled/cancelled workflow type pickers
  • Additional Details: Free-text details field, custom attributes

Schedule Picker

[Placeholder: Screenshot of the schedule card picker with multiple selected]

Replaces the standard multi-select list box with a card-based checkbox grid. Each card shows the schedule name, HTML description, friendly recurrence text, next occurrence date, category, slot capacity, and duration. Cards are sorted by next start date (upcoming first).
