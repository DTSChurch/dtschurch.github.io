Lava Filters

Custom Lava filters for querying appointment data in any Lava template — system communications, HTML blocks, mobile pages, or workflow entry forms.

AppointmentById

Returns a single Appointment entity by its integer Id.

{{ 42 | AppointmentById }}

Input

Integer or string-parsable appointment Id.

Returns

An Appointment entity with full navigation properties:

  • .PersonAlias.Person — the person who booked
  • .AppointmentType — the appointment type configuration
  • .Schedule — the appointment schedule
  • .AppointmentDate — the booked date/time
  • .AttendanceStatus — Pending, Attended, or Cancelled
  • .BookingNotes, .CancellationNotes

Example

{% assign appointment = Workflow.AppointmentId | AppointmentById %}
{% if appointment %}
    <p>{{ appointment.PersonAlias.Person.FullName }} has an appointment
    on {{ appointment.AppointmentDate | Date:'dddd, MMMM d \a\t h:mm tt' }}
    for {{ appointment.AppointmentType.Name }}.</p>
{% endif %}

AppointmentTypeById

Returns a single AppointmentType entity by its integer Id.

{{ 5 | AppointmentTypeById }}

Input

Integer or string-parsable appointment type Id.

Returns

An AppointmentType entity with:

  • .Name, .AppointmentTerm
  • .Campus — full Campus entity
  • .Location — full Location entity
  • .Category — full Category entity
  • .CanCancelOrEdit, .HoursToChange, .WeeksAhead
  • .ConfirmationSystemEmail, .ReminderSystemEmail
  • .WorkflowType, .RescheduledWorkflowType

Example

{% assign type = 5 | AppointmentTypeById %}
<h3>{{ type.Name }}</h3>
<p>Campus: {{ type.Campus.Name }} | Location: {{ type.Location.Name }}</p>
<p>Book up to {{ type.WeeksAhead }} weeks ahead.</p>

AppointmentScheduleById

Returns a single AppointmentSchedule entity by its integer Id.

{{ 10 | AppointmentScheduleById }}

Input

Integer or string-parsable schedule Id.

Returns

An AppointmentSchedule entity with:

  • .Name, .Description
  • .FriendlyScheduleText — human-readable recurrence
  • .AvailableSlots — slots per occurrence (0 = unlimited)
  • .SlotDurationMinutes, .BufferTimeMinutes
  • .EffectiveStartDate, .EffectiveEndDate
  • .Category — full Category entity

Example

{% assign schedule = 10 | AppointmentScheduleById %}
<p>{{ schedule.Name }}: {{ schedule.FriendlyScheduleText }}</p>
<p>{{ schedule.AvailableSlots }} slots per occurrence</p>

AppointmentsByTypeId

Returns all appointments for a given appointment type, ordered by date descending.

{{ 5 | AppointmentsByTypeId }}

Input

Integer or string-parsable appointment type Id.

Returns

A list of Appointment entities (newest first). Includes all statuses.

Example

{% assign appointments = appointmentTypeId | AppointmentsByTypeId %}
<h4>Recent Bookings ({{ appointments | Size }})</h4>
<ul>
{% for apt in appointments limit:10 %}
    <li>{{ apt.PersonAlias.Person.FullName }}
        &mdash; {{ apt.AppointmentDate | Date:'MMM d, yyyy h:mm tt' }}
        ({{ apt.AttendanceStatus }})</li>
{% endfor %}
</ul>

UpcomingAppointments

Returns upcoming (non-cancelled) appointments for a person, ordered by date ascending. Accepts a Person object or a person Id. 

{{ CurrentPerson | UpcomingAppointments }}

Input

A Rock Person entity or an integer person Id.

Returns

A list of Appointment entities where the appointment date is in the future and the status is not Cancelled. Ordered by appointment date ascending (soonest first).

Example — Person Profile Badge

{% assign upcoming = CurrentPerson | UpcomingAppointments %}
{% if upcoming != empty %}
    <span class="badge badge-info">
        {{ upcoming | Size }} upcoming {{ 'appointment' | PluralizeForQuantity:upcoming.Size }}
    </span>
{% endif %}

Example — My Appointments Page

{% assign upcoming = CurrentPerson | UpcomingAppointments %}
{% if upcoming != empty %}
    <div class="panel panel-default">
        <div class="panel-heading"><h3>Your Upcoming Appointments</h3></div>
        <div class="panel-body">
            {% for apt in upcoming %}
                <div class="d-flex justify-content-between align-items-center py-2 border-bottom">
                    <div>
                        <strong>{{ apt.AppointmentType.Name }}</strong>
                        <br><small class="text-muted">{{ apt.AppointmentDate | Date:'dddd, MMMM d, yyyy \a\t h:mm tt' }}</small>
                        <br><small class="text-muted">{{ apt.AppointmentType.Location.Name }}</small>
                    </div>
                    <div>
                        <a href="/book-appointment?Appointment={{ apt.Guid }}" class="btn btn-xs btn-default">Edit</a>
                        <a href="/book-appointment?Appointment={{ apt.Guid }}&Cancel=true" class="btn btn-xs btn-danger">Cancel</a>
                    </div>
                </div>
            {% endfor %}
        </div>
    </div>
{% else %}
    <p class="text-muted">You have no upcoming appointments.</p>
{% endif %}
