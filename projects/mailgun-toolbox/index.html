Mailgun Toolbox

Email monitoring dashboard for Rock RMS

The Mailgun Toolbox plugin provides an email health dashboard inside Rock RMS. Monitor delivery rates, manage bounces and complaints, drill into failure details, and analyze per-domain statistics — all without leaving Rock.

Get Started Setup Guide

What you get

  • Overview tab: See delivery, open, click, and bounce rates at a glance with daily trend charts.
  • Details tab: Investigate delivery failures — see error breakdowns and the most recent failed messages.
  • Bounces tab: View and manage the list of bounced email addresses. Remove addresses to allow re-delivery.
  • Historicals tab: Explore email volume over time at hourly, daily, or monthly resolution.
  • Domains tab: See how email performs per recipient domain (gmail.com, outlook.com, etc.).

A date range filter applies across all tabs. You can choose from preset ranges or enter custom dates.

Requirements

  • Rock RMS v16.9+ (Obsidian)
  • Mailgun account with API access
  • Mailgun configured as your Rock email transport

Getting started

  1. Install the plugin — find it in the Rock Shop and install.
  2. Configure your credentials — set your API key and sending domain.
  3. Explore the dashboard — learn what each tab shows and what to look for.
  4. Having trouble? — common issues, bounce rate guidance, and how to get support.
